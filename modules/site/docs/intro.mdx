## Installation

:::info

Scala.php only supports Scala 3.3.0 and above.

:::

<Tabs>
<TabItem value='sbt'>

In your `project/plugins.sbt`:

```scala title="project/plugins.sbt" copy
addCompilerPlugin("org.polyvariant" % "scala-php-sbt" % "0.1.0" cross CrossVersion.full)
```

Now, you can enable the plugin on the modules you want to compile to PHP:

```scala title="build.sbt" copy
val main = project
  .settings(scalaVersion := "3.3.1")
  .enablePlugins(PhpPlugin)
```
